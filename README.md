# Worldwide News Map

## How to Run

### 1. Install Node.js

Download and install Node.js from:

https://nodejs.org/

Check that it is installed:

```bash
node -v
npm -v
```

---

### 2. Clone the repository

```bash
git clone git@github.com:Nikolai-Marinov/Worldwide-News.git
```

Open the project folder:



---

### 3. Install dependencies

```bash
npm install
```

---

### 4. API Key Setup

This project uses NewsAPI.

Create an account at NewsAPI and get your API key.
Create a new file called:
.env
Add your API key:
NEWS_API_KEY=your_api_key_here


### 5. Start the server

```bash
node server.js
```

Keep the terminal open.

---

### 6. Open the website

Open `page.html` using Live Server in VS Code.

The map and news features will now work.
